How it works

The installer keeps the reading experience boring in the right way: stock Kobo, stock library, normal EPUB files.

01

Fetch Matter queue

The sync client reads queued Matter articles through the public API.

02

Write EPUBs

Articles become ordinary EPUB files under the visible Matter folder.

03

Rescan library

NickelDBus asks Kobo's stock reader to import the new books.

Built for a real Kobo, not a replacement OS

Kobo plus Matter uses stock firmware, NickelMenu, and NickelDBus. It does not install KOReader, replace Nickel, or ask you to change where you read.

  • Manual and automatic sync modes
  • Visible NickelMenu actions for sync, status, Wi-Fi, and import
  • Optional Matter collection labeling after import
  • Recovery path documented for uninstall and token changes

Try the current package

The release artifact is a Kobo updater package. Copy it to the hidden .kobo folder, eject, and let the device reboot.
